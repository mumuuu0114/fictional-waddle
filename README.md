# Project Assignment

## Project Title

Simulation and Analysis of the Motion of a Ball Rolling Off a Table

## Overview

This repository contains the Python code used for the general physics project assignment.
The purpose of this project is to simulate the motion of a ball rolling off a table and to compare the computational simulation results with the theoretical model and experimental observations.

In this project, the motion of the ball is analyzed using basic principles of mechanics, including translational motion, rotational motion, projectile motion, and energy conservation. The simulation was used to visualize the motion of the ball and to support the analysis presented in the project report.

## Project Description

The project is based on the motion of a ball that rolls along a table and then leaves the edge. Before the ball leaves the table, it undergoes rolling motion. After leaving the table, the ball follows projectile motion under the influence of gravity.

The simulation code calculates and visualizes the motion of the ball using the physical model discussed in the project. The computational result helps us understand the relationship between the theoretical prediction, experimental result, and numerical simulation.

## Simulation Method

The simulation models the motion of the ball in two main stages.

First, the ball rolls on the table. In this stage, the motion is related to rolling motion and rotational dynamics.

Second, after the ball leaves the edge of the table, the ball follows projectile motion. In this stage, the horizontal velocity remains approximately constant while the vertical motion is affected by gravitational acceleration.

The numerical calculation is used to predict the trajectory of the ball, and the result is compared with the theoretical and experimental results discussed in the report.

## Purpose of the Code

The code was written to support the project report by providing a computational simulation of the physical system.

Through the simulation, we can:

* visualize the motion of the ball,
* compare theoretical predictions with computational results,
* analyze the relationship between rolling motion and projectile motion,
* and better understand the physical principles discussed in the project.
